
# MScWebTecProject NodeJS API


## Install Dependencies

```bash
  npm install
```


## Run the Application

```bash
 node app.js
```

## Necessary Database Commands

Create database

```bash
  npm run db-create
```

Drop database

```bash
  npm run db-drop
```

Create all tables

```bash
  npm run migrate
```

Drop all tables

```bash
  npm run migrate-undo-all
```

Drop all tables then create again & insert pre defined data

```bash
  npm run fresh-seed
```
